This is a monorepo for the open-sourced project 'Minstra'

It contains the following packages:

- packages/client: The client application
- packages/server: The server application

To run the project, use the following commands:

- npm run dev:client: Starts the client in development mode
- npm run dev:server: Starts the server in development mode
- npm run build:client: Builds the client
- npm run build:server: Builds the server
- npm run start:client: Starts the client
- npm run start:server: Starts the server
